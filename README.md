# ⚡ Trello On-the-Fly Extension

A lightweight browser extension that supercharges your Trello workflow with fast, on-the-fly actions and productivity shortcuts.

Built for speed, simplicity, and minimal UI — no clutter, just efficiency.

---

## ✨ Features

* ⚡ Quick actions directly inside any active tab
* 🧩 Seamless integration with Trello boards
* 🎯 Minimal and distraction-free design
* 🚀 Boost productivity without leaving any website 

---

## 💡 Why this extension?

Trello is powerful, but repetitive actions slow you down.

This extension removes friction by letting you:

* Perform actions instantly
* Reduce clicks
* Stay focused on your tasks

---

## 🛠️ Tech Stack

* JavaScript 
* Trello  API
* DOM Manipulation

---

## 📦 Installation

1. Clone the repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the project folder
( You need your own Trello API)

---

## 🚀 Usage

1. Connect your Trello account
2. Open any website
3. Enjoy faster workflow ⚡
